# 染色背包
黏液科技的附加 繁體翻譯版<br>

可以讓你的背包染成您喜愛的顏色
- 使用羊毛與染料將所有背包染成你想要的顏色!
- 支持所有16種Minecraft默認顏色.

此為**非官方**版本, 請勿在該作者的問題追蹤內回報!
[原作者連結](https://github.com/TheBusyBiscuit/DyedBackpacks)
